# Automatisation-d-envoi-de-formulaire-avec-Playwright-en-Python
Script Python utilisant Playwright pour automatiser l’envoi de formulaires web via requêtes POST. Permet de tester et déboguer l’envoi de données, gérer les headers et cookies, et analyser les réponses JSON. Compatible Chromium, Firefox et WebKit. Pré-requis : Python 3.9+, Playwright et navigateurs installés.
